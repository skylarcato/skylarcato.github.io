“Programming languages for creating web pages.”

Web pages are documents that displays in web browser. JavaScript, HTMl, CSS and PHP are the most common programming languages for creating web pages. Brackets.io, Notepad etc are the available editors for writing and editing the codes.

About HTML

HTML stands for Hyper Text Markup language. HTML uses tags to instruct the web browser on how to display the contents.
